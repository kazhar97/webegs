== Installation ==

Installation process of this plugin is very easy.

<blockquote>
     <strong>Option 1</strong>
    <ul>
        <li>Download the plugin image-hover-effects-with-carousel.zip from WordPress Plugin Directory.</li>
        <li>Unzip the zip file and extract the plugin folder named <strong>image-hover-effects-with-carousel</strong></li>
        <li>Left that plugin folder into your wp-content/plugins folder.</li>
        <li>Go and refresh your Wordpress administration panels and from the menu bar click on Plugins.</li>
        <li>Now you may see your plugin listed under inactive plug-in tab.</li>
        <li>Click active to turn your wordpress plugin on.</li> 
    </ul>

</blockquote>

<blockquote>

    <strong>Option 2</strong>
    <ul>
        <li>Like the first option you have to download the zip file from download location.</li>
        <li>Unzip the zip file and extract the image-hover-effects-with-carousel plugin folder.</li>
        <li>Double click on the installable folder in order to get image-hover-effects-with-carousel.zip.</li>
        <li>Go to Wordpress admin panel and press Add New from the Menu bar under plugins tab.</li>
        <li>Select upload link and drug the available image Hover Effects Plugin file and click install now.</li>
        <li>After successfully installed finally click on ‘activate’ to turn on.</li>
    </ul>

</blockquote>

That’s all about the installation process. For further query you’re welcome to contact us
